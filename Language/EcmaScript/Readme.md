
# EcmaScript6(Ecma2015)

## 소개</br>
대부분의 자바스크립트는 2009년에 제정되어 2011년에 개정된 EcmaScript5.1을 표준으로 사용함.
지금은 ECMA라는 단체에서 매년 결점을 보완하여 표준 자바스크립트 버전을 발표함. 2015년 6월에 대대적인 업데이트를 통해 EcmaScript6가 발표되었음.
